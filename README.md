# Benjamin Adedowole Portfolio

This branch contains a clean-room React recreation of a macOS-inspired portfolio interface.

## Stack

- React 18
- Vite
- Tailwind CSS
- Custom CSS for the desktop, windows, menu bar, dock, and icons

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Build

```bash
npm run build
npm run preview
```

## Notes

The dock icons and wallpaper in this branch are original SVG assets stored under `public/`. The app content is stored in `src/data/portfolio.js` so the name, project links, skills, and resume/contact links are easy to edit.
